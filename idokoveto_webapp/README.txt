Időkövető Webapp
=====
Ez a projekt egy egyszerű, böngészőben futó időkövető alkalmazás. Főbb funkciói:

- Feladatok kezelése
- Pomodoro időzítő
- Napi/heti statisztikák megjelenítése (Chart.js)
- CSV exportálás
- Időzített emlékeztetők

Frontend: HTML, CSS, JS
Backend (ötletként): F# + .NET + SQLite / JSON
